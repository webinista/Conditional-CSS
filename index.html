What is Conditional CSS?

A new way of thinking about some of the things we already know and use. Also a CSS Level 3 module, the Conditional CSS Rules module.

What is Conditional CSS?

CSS features and JavaScript APIs that let us specify CSS style rules that are only applied when the condition is met.

Conditional CSS: the @rules

  • @media
  • @page
  • @supports
  • @viewport
  • @document

Includes our old friends, @media and @page, which were part of the CSS Level 2 specification.

Newer features and APIs

  • Media queries, which extend the @media rules.
  • @supports, which offers native CSS feature detection
  • @viewport, which applies styles based on the document's viewport
  • @document, which applies styles based on the document's URL

@media: a recap

Set styles for a particular media type.

@media print{
    body { font-size: 10pt }
}
@media screen, print {
    body { line-height: 1.2 }
}
